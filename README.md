### [🎙️ AI Voice Agent Infrastructure Platform](https://github.com/sanjanamahajan2001-sys/AI-Voice-Infrastructure-Platform)
*A production-ready infrastructure platform for deploying scalable AI voice/chat agents.*
- **Compute:** AWS EKS (Managed Kubernetes)
- **IaC:** Terraform (Modular multi-AZ VPC, RDS, EKS)
- **CI/CD:** GitHub Actions (Automated build, test, and deploy)
- **Observability:** Prometheus, Grafana, and Loki stack
- **Key Feature:** High-concurrency WebSocket streaming support with horizontal autoscaling.

### [⚡ Kubernetes Deployment Accelerator](https://github.com/sanjanamahajan2001-sys/k8s-deployment-accelerator)
*A "Platform-in-a-Box" designed to reduce app deployment setup time by 70%.*
- **Infra:** Modular Terraform modules for rapid EKS provisioning.
- **Packaging:** Standardized Helm Library Charts for microservices.
- **Automation:** GitHub Actions & Bitbucket Pipelines for GitOps delivery.
- **Security:** OPA Gatekeeper policies and Trivy vulnerability scanning.

### [🔍 Production Monitoring & Incident Platform](https://github.com/sanjanamahajan2001-sys/production-monitoring-platform)
*Centralized observability stack for proactive infrastructure reliability.*
- **Metrics:** High-resolution Prometheus metrics and AlertManager routing.
- **Logging:** Distributed log aggregation with Grafana Loki.
- **Alerting:** Automated Slack notifications for CPU, Memory, and SLO breaches.
- **Root Cause:** Unified Grafana dashboards linking logs directly to metrics.
